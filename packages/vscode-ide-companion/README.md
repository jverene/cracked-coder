# Cracked Coder Companion

The Cracked Coder Companion extension pairs with
[Cracked Coder](https://github.com/cracked-coder/cracked-coder). This extension is
compatible with both VS Code and VS Code forks.

# Features

- Open Editor File Context: Cracked Coder gains awareness of the files you have
  open in your editor, providing it with a richer understanding of your
  project's structure and content.

- Selection Context: Cracked Coder can easily access your cursor's position and
  selected text within the editor, giving it valuable context directly from your
  current work.

- Native Diffing: Seamlessly view, modify, and accept code changes suggested by
  Cracked Coder directly within the editor.

- Launch Cracked Coder: Quickly start a new Cracked Coder session from the Command
  Palette (Cmd+Shift+P or Ctrl+Shift+P) by running the "Cracked Coder: Run"
  command.

# Requirements

To use this extension, you'll need:

- VS Code version 1.99.0 or newer
- Cracked Coder (installed separately) running within the integrated terminal
